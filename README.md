# Calculator App

A simple iOS-style calculator built with Flutter as one of my early projects.

## Features
- Basic arithmetic: `+` `-` `*` `/` `%`
- Backspace and full reset
- Clean dark UI

## Getting Started
```bash
flutter pub get
flutter run
```

## Dependencies
- [math_expressions](https://pub.dev/packages/math_expressions)
